Detailed Description

Support module that creates a moving average filter for data.

Use is very easy. Construct an instance of the class specifying the number of data points to be used in the filter. Pass in new data points with the *.process method. Method returns updated moving average filtered value. Found at: https://github.com/sebnil/Moving-Avarage-Filter&ndash;Arduino-Library-
